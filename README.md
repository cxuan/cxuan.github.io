# 这是什么？
这是[博客网站](http://c-xuan.com)的站点，由[Hexo](https://hexo.io/)生成的静态网页，部署到[github pages](https://pages.github.com/)，无须购买服务器，可以绑定自己的域名，使用Markdown撰写博客文章，适合折腾。

# 网站修改历史
- 2016-08-20:站点首次提交到github.
- 2016-09-25:
  - 旧博客迁移,放弃使用[B3log-solo](http://b3log.org/).
